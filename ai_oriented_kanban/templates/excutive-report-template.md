## Writing Guidelines (for non-technical audiences)

- Keep paragraphs short (2–4 lines).
- Prefer outcomes over implementation detail.
- Use concrete before/after comparisons.
- Use numbers wherever possible (time, cost, incidents, conversion, effort).
- Avoid jargon; if needed, define in one line.
- End with a clear recommendation.
